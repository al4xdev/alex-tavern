# 📱 Mobile Gestures

Optimize your gaming experience on mobile with these gestures:

## Scene Panel Toggle
- When the scene has many description lines or physical tags, a swipe gesture is supported.
- **Swipe Down** on the Scene Panel: expands it to show full details.
- **Swipe Up** on the Scene Panel: collapses it back to a compact view.
- **Tap**: toggles expansion directly.

## Input Area Gestures
Swipe gestures are available on the bottom input panel (collapsed or expanded bar):
- **Swipe Right** (slide from left to right): triggers **Undo last turn** ↩.
- **Swipe Left** (slide from right to left): opens the **Narrator event hint popup** 📜 (with auto-skip).
- **Swipe Down**: collapses the input panel.
- **Swipe Up**: expands the input panel to type speech, thoughts, and actions.

## Chat Scrolling
- The chat log scrolls automatically to the latest events, but you can manually scroll up to review previous turns at any time.

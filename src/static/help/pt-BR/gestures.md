# 📱 Gestos no Celular

Otimize sua experiência de jogo no celular com estes gestos:

## Expandir/Recolher Painel de Cena
- Quando a cena atual tiver muitas linhas de descrição ou tags físicas, você pode usar gestos de deslizamento.
- **Deslizar para Baixo** no Painel de Cena: expande o painel para exibir os detalhes completos.
- **Deslizar para Cima** no Painel de Cena: recolhe o painel de volta para a visualização compacta.
- **Tocar**: alterna a expansão diretamente.

## Rolagem do Chat
- O chat rola automaticamente até os últimos eventos do jogo, mas você pode rolar manualmente para cima para revisar turnos anteriores a qualquer momento.
